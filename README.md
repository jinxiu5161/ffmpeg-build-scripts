# ffmpeg的编译
```
git clone https://github.com/jinxiu5161/ffmpeg-build-scripts.git
cd ffmpeg-build-scripts
```
1、for iOS
```
./compile-ios.sh
```
2、for android
```
./compile-android.sh
```
3、for mac
```
./compile-pc.sh mac
```
4、for ubuntu
```
./compile-pc.sh linux
```
5、for windows 64
```
./compile-pc.sh windows
```
